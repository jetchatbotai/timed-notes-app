# Timed Notes

A simple multilingual note-based timer app.

## Features

- Add a note
- Set hours, minutes and seconds
- Start a timer
- Background alarm
- Local notifications
- Multilingual support

## Languages

- English
- Turkish
- German
- French
- Spanish
- Italian
- Portuguese
- Arabic
- Russian
- Japanese
- Hindi
- Chinese

## Status

Project in development.
